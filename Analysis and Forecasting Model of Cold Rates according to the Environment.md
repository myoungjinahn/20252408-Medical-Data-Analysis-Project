# 서울시의 환경에 따른 감기 발생률 관계 분석,예측 모델 프로젝트 제안서

---

## 1. 프로젝트 개요

**프로젝트 이름**: 서울시 감기 발생과 환경 요인의 관계 분석  
**작성자**: 안명진  
**작성일**: 2026년 4월 12일  
**목적**: 기상 및 대기오염 요인이 서울시 감기 발생에 미치는 영향을 분석하고, 이를 기반으로 월별 발생률을 예측하는 모델 구축

- **주요 알고리즘**: 다중 선형 회귀 (기준 모델), Random Forest Regressor (예측 모델)

## 2. 배경 및 필요성

작성자의 경험으로 평균기온이 낮고 연교차가 10도 이내일 때에서는 감기에 걸리지 않다가, 평균기온이 높아지고 연교차가 15도 이상일 때 감기에 걸리는 상황을 겪었으며, 또한 미세먼지가 감기 발병률에 큰 영향을 미친다는 연구를 보고 실제적으로 미세먼지 농도와 발병률이 어떤 상호작용을 하는지 확인하고 싶어 **감기발생과 환경요인의 관계 분석 모델을** 제작하게 되었다

## 4. 목표

기상 및 대기오염 요인이 서울시 감기 발생에 미치는 **영향력을 정량적으로 파악**하고, 이를 기반으로 **월별 감기 발생 수를 예측**하는 것

- 기상 요인(기온, 습도)과 대기오염 요인(PM10, PM2.5)이 감기 발생 수에 미치는 영향력을 수치로 파악
- 계절별 환경 요인과 감기 발생 간의 패턴 차이를 시각화하여 인사이트 도출
- 분석된 영향 요인을 기반으로 월별 감기 발생 수를 예측하는 머신러닝 모델 구축
- 예측 결과와 실제 데이터를 그래프로 비교하여 모델의 신뢰도 검증

## 5. 주요 기능 및 특징

### 5-1. 환경 요인 영향 분석 (상관분석)
- 기온, 습도, PM10, PM2.5 각 변수와 감기 발생 수 간의 **Pearson / Spearman 상관계수** 산출
- 변수 간 상관관계를 Heatmap으로 시각화하여 영향력이 큰 요인 식별

### 5-2. 계절별 발생 패턴 분석
- 봄·여름·가을·겨울 4개 계절을 그룹 변수로 분류
- 계절별 감기 발생 분포와 환경 요인 수치 변화를 그래프로 비교분석

### 5-3. 다중 회귀 분석
*모델 선정 이유:프로젝트의 핵심 목표가 "영향 요인 분석" 이기 떄문에. 기온이 1도 내려가면 감기 환자가 얼마나 늘어나는지, PM2.5의 수치 변동이 얼마나 영향을 주는지를 숫자로 직접 보여줄 수 있는 모델이 선형 회귀이기 때문*

- **다중 선형 회귀**로 각 환경 변수의 회귀 계수와 통계적 유의성(p-value) 검정
- 회귀 계수의 방향(+/-)과 크기로 각 요인의 영향 방향 및 강도 해석


### 5-4. 발생률 예측 모델 구축 및 평가
*실제 감기 발생은 기온이 특정 구간 이하로 떨어질 때 급격히 증가하는 비선형 패턴이 있어. 선형 회귀 모델로 처리하기 부적합하다고 판단, Random Forest는 이걸 잡아낼 수 있고, Feature Importance 그래프로 어떤 환경 요인이 예측에 제일 중요했는지 시각화할 수 있어서 "영향 요인이 드러나는 그래프" 목표에 적합하다고 판단함*

- **Random Forest Regressor**로 월별 감기 발생 수 예측
- Feature Importance 그래프로 예측에 기여한 환경 요인의 중요도 시각화
- 아래 3가지 지표로 모델 성능을 측정

| 지표 | 설명 |
|------|------|
| **MAE** (평균 절대 오차) | 예측값과 실제값 차이의 평균 |
| **RMSE** (평균 제곱근 오차) | 큰 오차에 더 민감한 성능 지표 |
| **R²** (결정계수) | 모델이 실제 데이터 분산을 설명하는 비율 |
